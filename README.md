# WDD231 Project

## Overview
This project is a web development course portfolio for the WDD231 class. It showcases the skills and knowledge acquired throughout the course, including responsive design, semantic HTML, CSS styling, and JavaScript functionality.

## Project Structure
```
wdd231
├── index.html          # Main HTML document serving as the home page
├── images              # Folder containing optimized images
│   ├── logo.svg       # Site logo in SVG format
│   └── (optimized-content-images...) # Other optimized images
├── styles              # Folder containing CSS files
│   ├── normalize.css   # CSS reset for consistent styling across browsers
│   ├── small.css       # Custom styles for small screen sizes
│   └── large.css       # Custom styles for larger screen sizes with media queries
├── scripts             # Folder containing JavaScript files
│   ├── navigation.js   # JavaScript for responsive navigation menu
│   ├── date.js         # JavaScript for displaying current year and last modified date
│   ├── courses.js      # JavaScript for handling course information display
│   └── course-data.js  # Array of course objects for dynamic display
├── README.md           # Documentation for the project
└── .gitignore          # Files and directories to be ignored by Git
```

## Features
- Responsive design that adapts to various screen sizes.
- Semantic HTML structure for better accessibility and SEO.
- Dynamic display of the current year and last modified date using JavaScript.
- Course information displayed dynamically with filtering options for completed and uncompleted courses.

## Setup Instructions
1. Clone the repository from GitHub.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to view the project.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- SVG for images
